Optionen

  • PIN voreinstellen
    Wählen Sie diese Option, wenn Sie die Konferenz mit einem allgemeinen PIN schützen möchten. Geben Sie dazu die PIN in das Feld PIN ein.
  • Konferenz aufnehmen

    Wählen Sie diese Option, wenn Sie die Konferenz aufzeichnen möchten. Wählen Sie dazu aus der Liste das gewünschte Format für die Aufnahme.

  • Wartemusik bei Halten
    Wählen Sie diese Option, wenn der oder die gehaltenen Anrufer eine Wartemusik hören sollen.
  • Sitzungsmenü aktivieren

    Wählen Sie diese Option, wenn Sie das Sitzungsmenü aktivieren möchten.

  • Benachrichtige über Zugang/Abgang von Teilnehmern

    Wählen Sie diese Option, um jedesmal, wenn ein Teilnehmer die Konferenz betritt oder verlässt eine Benachrichtigung abzuspielen.

  • Zähle Benutzer

    Wählen Sie diese Option, um die Anzahl der Teilnehmer zu zählen.
