Ein neues System erstellen


Um ein neues System zu erstellen dienen die folgenden Knöpfe oberhalb der Liste:

  • zum Erstellen eines neues Terminals,
  • zum Erstellen einer neuen Arbeitsstation,
  • zum Erstellen eines neuen Servers,
  • zum Erstellen eines neuen Druckers,
  • zum Erstellen eines neuen Telefons,
  • zum Erstellen einer neuen Netzwerkkomponente.
Wenn Sie ein vorhandenes System bearbeiten möchten, klicken Sie auf den Namen des Systems oder auf den Knopf .
