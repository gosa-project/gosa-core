Optionen

Der Reiter Optionen dient dazu, Umgebungsvariablen für Anwendungen zu setzen. Standardmässig sind Optionen deaktiviert und müssen zunächst mit einem Klick auf den Knopf Optionen aktivieren hinzugefügt werden. Die verfügbaren Felder auf der Seite sind in der folgenden Tabelle beschrieben:

Variable Name der Variablen
Standardwert Der Standardwert der Variablen (kann überschrieben werden)


Um eine neue Variable hinzuzufügen, verwenden Sie den Knopf Option hinzufügen. Um eine vorhanden Variable zu entfernen, verwenden Sie den Knopf Entfernen.
