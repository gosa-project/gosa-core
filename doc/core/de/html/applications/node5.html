Skript

Der Inhalt dieser Textbox wird beim Start der Anwendung ausgeführt. Sie können entweder das Skript direkt in der Textbox editieren, oder ein vorhandenes Skript hochladen (Dazu klicken Sie auf den Knopf rechts neben der Eingabezeile unterhalb der Textbox - dieser ist je nach Browser unterschiedlich - und wählen das gewünschte Skript aus. Um das gewählte Skript in das Fenster zu laden, klicken Sie auf den Knopf Hochladen.

Beispiele für die Verwendung eines Skriptes:

  • Erstellen von Verzeichnissen
  • Setzen von Konfigurationsparametern
