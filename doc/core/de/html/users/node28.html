Zugriffsoptionen

Die unter Zugriffsoptionen aufgeführten Einstellung dienen zum Anpassen der Passwort-Einstellungen. Sie sollten genau aufpassen, welche Einstellungen Sie vornehmen.

  • Die folgenden Optionen sind verfügbar:

    • Der Benutzer darf das Passwort vom Client aus ändern: Wenn diese Option aktiv ist, darf der Benutzer vom Windows-Client sein Passwort ändern.
    • Die Anmeldung vom Windows-Client erfordert kein Passwort: Wenn diese Option aktiv ist, benötigt der Benutzer kein Passwort, um sich anzumelden.
    • Samba-Konto sperren: Wenn diese Option aktiv ist, kann der Benutzer sich nicht mehr verbinden.
    • Passwort läuft ab am: Wählen Sie ein Datum, an dem das Passwort abläuft, das Konto wird dann automatisch gesperrt.
    • Limitiere Logon-Zeit: Zu dieser Einstellung sind keine Informationen verfügbar (sambaLogonTime).
    • Limitiere Logoff-Zeit: Zu dieser Einstellung sind keine Informationen verfügbar (sambaLogoffTime).
    • Konto läuft ab am: Wählen Sie ein Datum, an dem das Konto automatisch gesperrt wird.
  • Erlaube Verbindungen nur von diesen Arbeitsstationen: Wenn Sie den Zugriff des Benutzers auf eine bis mehrere Arbeitsstationen beschränken möchten, tragen Sie diese in die Liste ein. Wenn die Liste leer ist, ist der Zugriff von allen Clients möglich.

    • Hinzufügen: Um eine Arbeitsstation hinzufügen, klicken Sie auf den Knopf Hinzufügen und wählen Sie eine oder mehrere Arbeitsstationen aus der Liste, indem Sie sei markieren. Wenn Sie mit der Auswahl zufrieden sind, drücken Sie den Knopf Hinzufügen. Die gewünschten Arbeitsstation werden nun in der Liste aufgeführt.
    • Entfernen: Um eine oder mehrere Arbeitsstationen aus der Liste zu entfernen, markieren Sie den oder die gewünschten Einträge in der Liste und drücken auf den Knopf entfernen. Die markierten Arbeitsstationen werden nun nicht mehr in der Liste aufgeführt.
