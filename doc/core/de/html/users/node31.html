FTP Konto

Mit diesem Konto können Sie dem Benutzer den Zugriff auf den FTP-Server erlauben:

  • Bandbreite: Erlaubt Einschränkung der Bandbreite (Aus Sicht des Benutzers)

    • Upload-Bandbreite: Die maximale Bandbreite für den Upload (Senderichtung). Wenn Sie 0 eintragen, ist die Beschränkung nicht aktiv.
    • Download-Bandbreite: Die maximale Bandbreite für den Download (Empfangsrichtung). Wenn Sie 0 eintragen, ist die Beschränkung nicht aktiv.
  • Verhältnis: Der Benutzer kann gezwungen werden, ein Verhältnis von Up- und Download von Dateien einzuhalten.

    • Hoch- / heruntergeladene Dateien: Stellen Sie das Verhältnis in Dateien ein (z.B. 4/1: 4 Dateien müssen hochgeladen werden, damit der Benutzer eine Datei herunterladen darf).
  • Kontingent: Einschränkung des Kontingents in Dateien oder übertragenen Datenmengen

    • Dateien: Die maximale Anzahl an Dateien, die der Benutzer übertragen darf. Wenn Sie 0 eintragen, ist die Beschränkung nicht aktiv.
    • Größe: Die maximale Datenmenge, die der Benutzer übertragen darf. Wenn Sie 0 eintragen, ist die Beschränkung nicht aktiv.
  • Verschiedenes:

    • Temporäres Abschalten des FTP-Zugriffs: Mit dieser Option können Sie den Zugriff auf den FTP-Server deaktivieren, ohne dabei die vorgenommenen Einstellungen zu verlieren.
