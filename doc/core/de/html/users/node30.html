Proxy Konto

Mit diesem Konto können Sie dem Benutzer den Zugriff auf den Proxy-Server erlauben:

  • Filtern von unerwünschten Inhalten (z.B. pornografische oder gewalttätige Inhalte): Diese Option aktiviert den Inhaltsfilter für den Benutzer. Dies hat zur Folge, dass keine Seiten aufgerufen werden können, die über fragwürdige Inhalte verfügen (bzw. vom Filter als solche erkannt werden).
  • Inhaltsfilterung nur während der Arbeitszeit: Die Inhaltsfilterung kann mit dieser Option ausschliesslich für die Arbeitszeit aktiviert werden (wählen Sie den Zeitraum). Ausserhalb der eingestellten Arbeitszeit ist der Zugriff uneingeschränkt.
  • Proxynutzung durch Kontingent einschränken: Mit dieser Option können Sie die maximal übertragene Menge für einen Zeitraum für den Benutzer festlegen. Wenn die Menge erreicht ist, kann der Benutzer keine Seiten mehr aufrufen.
