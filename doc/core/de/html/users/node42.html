Alternative Fax-Nummern

Es ist möglich, weitere Fax-Nummern für den Benutzer einzutragen. Alle alternativen Nummern, die dem Benutzer zugeordnet sind, werden in der Liste aufgeführt.

  • Hinzufügen: Um eine Nummer hinzuzufügen, geben Sie die Nummer in das Textfeld links neben dem Knopf ein und drücken Sie auf den Knopf Hinzufügen.
  • Lokale hinzufügen: Um eine Nummer hinzuzufügen, die bereits einem Benutzer zugeordnet wurde, drücken Sie auf den Knopf Lokale hinzufügen. Wählen Sie nun aus der Liste einen oder mehrere Benutzer, deren Faxnummern dem aktuell geöffneten Benutzer zugeordet wählen sollen, und drücken Sie den Knopf Hinzufügen. Um den Vorgang abzubrechen, drücken Sie auf den Knopf Abbrechen.
  • Entfernen: Um eine oder mehrere Nummern wieder zu entfernen, markieren Sie diese in der Liste (mit gedrückter STRG-Taste können Sie mehrere Nummern markieren) und drücken Sie den Knopf Entfernen. Die Nummern werden nun nicht mehr in der Liste aufgeführt.
