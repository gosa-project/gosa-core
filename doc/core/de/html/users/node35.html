Opengroupware

Mit dieser Option können Sie dem Benutzer den Zugriff auf den Opengroupware-Server erlauben:

  • Örtliches Team: Das örtliche Team, zu dem der Benutzer gehören wird.
  • Benutzer-Vorlage:
  • Gesperrt: Diese Option dient dazu, das Opengroupware-Konto zeitweilig zu sperren, ohne die Einstellungen zu verlieren. Wenn sie aktiv ist, kann der Benutzer sich nicht zum Opengroupware-Server verbinden.
  • Teams: Aktivieren Sie die Teams, denen der Benutzer angehören soll.
