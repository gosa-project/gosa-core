Liste der Sperrlisten

Die Liste der Sperrlisten dient als Ausgangspunkt für alle weiteren Schritte. Sie wird erreicht über den Menüeintrag Fax-Sperrlisten aus der Kategorie Administration (Menü am linken Rand des Bildschirms). Bei Auswahl wird die Liste der Sperrlisten geladen (Überschrift: Sperrlistenverwaltung). Auf dieser Seite können Sperrlisten hinzugefügt, bearbeitet oder entfernt werden.

Die Liste ist in zwei Spalten geteilt:

  • Die erste Spalte enthält die Namen der Sperrlisten
  • Die zweite Spalte enthält Knöpfe für die möglichen Aktionen, um Verwaltungsaufgaben durchzuführen (Bearbeiten, Entfernen)
Die vier Knöpfe , , , und oberhalb der Liste dienen zur Navigation innerhalb der Abteilungshierarchie:


  • : Zur Wurzel
  • : Eine Abteilung nach oben
  • : Zur Basis des angemeldeten Benutzers
  • : Aktuelle Abteilung neu laden
Es ist weiterhin möglich, die Anzeige der Sperrlisten mithilfe von Filtern zu beeinflussen (Kasten Filter am rechten Rand):

  • Nach Namen suchen :

    - Klick auf * zeigt alle Sperrlisten an

    - Klick auf einen Buchstaben zeigt alle Sperrlisten an, deren Name mit dem gewählten Buchstaben beginnt

    - Klick auf eine Zahl zeigt alle Sperrlisten an, deren Name mit der gewählten Ziffer beginnt

  • Schnellsuche : Suchkriterium (z.B. Name der Sperrlisten) eingeben, Klick auf Knopf Filter anwenden.


    Um eine neue Sperrliste zu erstellen, drücken Sie auf den Knopf oberhalb der Liste.

    Um eine bestehende Sperrliste zu bearbeiten, drücken Sie auf den Namen der Sperrliste
