Liste der Gruppen

Die Liste der Gruppen dient als Ausgangspunkt für alle weiteren Schritte. Sie wird erreicht über den Menüeintrag Gruppen aus der Kategorie Administration (Menü am linken Rand des Bildschirms). Bei Auswahl wird die Liste der Gruppen geladen (Überschrift: Gruppenverwaltung). Auf dieser Seite können Gruppen hinzugefügt, bearbeitet oder entfernt werden.

Die Liste ist in drei Spalten geteilt:

  • Die erste Spalte enthält zunächst die verfügbaren Abteilungen, dann die Namen der Gruppn (alphabetisch sortiert)
  • Die zweite Spalte enthält Knöpfe für den Schnellzugriff auf die verschiedenen Reiter der Gruppe (nur verfügbar, wenn die entsprechende Erweiterung aktiviert ist). Sie dient daher ausserdem für einen schnellen Überblick über die aktivierten Erweiterungen einer Gruppe

    • Die möglichen Symbole und ihre Bedeutung:
      Symbol Bedeutung
      Gruppe verfügt über UNIX-Eigenschaften
      Gruppe verfügt über Umgebungs-Einstellungen
      Gruppe verfügt über ein Mail-Konto
      Gruppe verfügt über SAMBA-Eigenschaften
      Gruppe verfügt über Anwendungs-Eigenschaften
  • Die dritte Spalte enthält Knöpfe für die möglichen Aktionen, um Verwaltungsaufgaben durchzuführen (Ausschneiden, Kopieren, Bearbeiten, Entfernen)
Die Knöpfe (, , , ) dienen zur Navigation innerhalb der Abteilungshierarchie:


  • Zur Wurzel
  • Eine Abteilung nach oben
  • Zur Basis des Benutzers
  • Aktuelle Abteilung neu laden
Es ist weiterhin möglich, die Anzeige der Gruppen mithilfe von Filtern zu beeinflussen (Kasten Filter am rechten Rand):

  • Nach Namen suchen:

    • Klick auf * zeigt alle Gruppen an
    • Klick auf einen Buchstaben zeigt alle Gruppen an, deren Namen mit dem gewählten Buchstaben beginnt
    • Klick auf eine Ziffer zeigt alle Gruppen an, deren Name mit der gewählten Ziffer beginnt
  • Weitere Suchoptionen:
    (Die folgenden Filter arbeiten so, dass nur Gruppen angezeigt werden, die über mindestens eine der ausgewählten Optionen verfügen; Standardmässig werden alle Gruppen angezeigt

    • Zeige primäre Gruppen: Zeigt Gruppen, die für mindestens einen Benutzer die primäre Gruppe ist
    • Zeige Samba-Gruppen: Zeigt Gruppen, die über Samba-Eigenschaften verfügen
    • Zeige Anwendungs-Gruppen: Zeigt Gruppen, die über Anwendungs-Eigenschaften verfügen
    • Zeige E-Mail-Gruppen: Zeigt Gruppen, die über ein Mail-Konto verfügen
    • Zeige Funktions-Gruppen: Zeigt Gruppen, die nur über die generischen Attribute verfügen
  • Komplexere Einschränkungen über reguläre Ausdrücke bieten die Textfelder im unteren Bereich des Kastens Filter:

    • Im ersten Feld kann nach Namen gesucht werden, die auf den eingebenen regulären Ausdruck passen.
    • Im zweiten Feld kann nach Gruppen anhand eines eingebenen Benutzerausdrucks gesucht werden, es werden also Gruppen gefunden, in denen ein auf den Suchausdruck passender Benutzer Mitglied ist.
