Weiterleiten der Nachrichten an nicht-Gruppenmitglieder

Mithilfe dieser Eigenschaften können Nachrichten, die an den gemeinsam genutzten Order geschickt werden, automatisch an weitere Adressen versendet werden. Geben Sie dazu in das Textfeld die Adresse ein, die Mails empfangen soll und drücken Sie auf den Knopf Hinzufügen. Wenn Sie einen oder mehrere Benutzer in die Liste aufnehmen möchten, die in dieser Organisation bereits über ein Mail-Konto verfügen, drücken Sie auf den Knopf Lokale hinzufügen. Wählen Sie den oder die gewünschten Benutzer aus der Liste, in dem Sie diese markieren und klicken Sie auf den Knopf Hinzufügen, um Sie in die Liste zu übernehmen.
