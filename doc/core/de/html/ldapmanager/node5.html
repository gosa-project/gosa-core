CSV Import

Auswahl der zu importierenden CVS Datei Drücken Sie auf den Knopf neben dem Textfeld, um die Dateiauswahl zu starten. Wählen Sie die CSV-Datei, die Sie importieren möchten.
Auswahl der Vorlage Wählen Sie die Vorlage, die verwendet werden soll, um nicht existente Werte (in der CVS-Datei) mit Werten zu füllen. Diese Option ist hilfreich, wenn Sie automatisiert sehr viele Benutzerkonten anlegen möchten.
