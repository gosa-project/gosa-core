Suche nach ZEICHENKETTE in WARTESCHLANGE mit Status: STATUS innerhalb der letzten ZEITBESCHRÄNKUNG:

ZEICHENKETTE Geben Sie die Zeichenkette ein, nach der sie suchen möchten.
WARTESCHLANGE Wählen Sie den Mail-Server aus der Liste, auf die Sie die Suche begrenzen möchten, oder Alle, wenn Sie alle Warteschlangen durchsuchen möchten.
STATUS Wählen Sie den Status aus der Liste, auf den Sie die Suche begrenzen möchten.
ZEITBESCHRÄNKUNG Wählen Sie die zeitliche Beschränkung aus der Liste, auf die Sie die Suche begrenzen möchten.

Zum Durchführen der Suche mit den vorgenommenen Einstellungen drücken Sie auf den Knopf Suchen ganz rechts.
