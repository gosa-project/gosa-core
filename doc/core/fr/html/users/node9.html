Système de confiance

Le système de confiance sert à gèrer l'accès de l'utilisateur aux différents systèmes, phériphériques, etc... . Le système de confiance peut être en mode :

  • désactivé,
  • accès complet : l'utilisateur à l'accès complet aux systèmes,
  • permettre l'accès à ces hôtes :

    - Utilisez le bouton Ajouter pour faire apparaître la liste des hôtes auxquels l'utilisateur pourrait avoir accès,

    - Utilisez le bouton Supprimer pour effacer les hôtes auxquels l'utilisateur n'a plus accès.
