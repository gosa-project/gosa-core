Alternative addresses

L'utilisateur peut être en possession d'alias. Ces alias sont à insérer dans le champ adresses alternatives.

Utilisez le bouton Ajouter pour insérer les alias. L'alias doit être introduit dans le champ vide précédant le bouton Ajouter. Utilisez le bouton Supprimer pour effacer les alias.
