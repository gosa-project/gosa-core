Optionen

  • Nur ausführbar für Gruppen-Mitglieder

    Wenn diese Option gewählt ist, dürfen nur Mitglieder einer entsprechenden UNIX- oder Objektgruppe die Anwendung ausführen (Es werden die Rechte der Datei unter Ausführen angepasst, daher ist diese Option nur sinnvoll, wenn die Anwendung nur einer Gruppe zugeordnet ist).

  • Konfiguration bei jedem Start ersetzen

    Wenn diese Option gewählt ist, wird bei jedem Start der Anwendung die vorhandene Konfiguration mit der Standardkonfiguration überschrieben.

  • Platziere das Symbol auf dem Desktop der Gruppenmitglieder

    Wenn diese Option gewählt ist, erscheint das Anwendungssymbol auf dem Desktop der Gruppenmitglieder.

  • Platziere einen Eintrag im Startmenü der Gruppenmitglieder

    Wenn diese Option gewählt ist, erscheint die Anwendung als Eintrag im Startmenü der Gruppenmitglieder.

  • Platziere einen Eintrag in der Kontrollleiste der Gruppenmitglieder

    Wenn diese Option gewählt ist, erscheint die Anwendung als Symbol in der Kontrollleiste der Gruppenmitglieder.
