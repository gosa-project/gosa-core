Auslieferungsmethode

  • Temporäres Abschalten der Fax-Benutzung: Deaktiviert das Fax-Konto des Benutzers, ohne die Einstellungen zu verlieren (Wenn Sie das Konto dauerhaft entfernen möchten, klicken Sie den Knopf Fax-Konto entfernen).
  • Fax als Mail ausliefern an: Die Mail-Adresse, an die das Fax im ausgewählten Format gesendet wird (Üblicherweise die Mailbox des Benutzers).
  • Fax an Drucker weiterleiten: Wählen Sie aus der Liste den Drucker, auf dem ein empfangenes Fax gedruckt wird.
